INFM600 - Information Organization Assignment

Dataset Title
==========================================================================
Liquor License, Vehicle Collision and Police Calls Over Howard County, MD


Version
==========================================================================
Version 1.0 (October 2015)


Description
==========================================================================
This dataset is the result of the combination of three other datasets; Howard County Police Call For Service, Vehicle Collision in Howard County and Active License of Alcohol and Tobacco Tax Payers in Howard County. To answer the research question, the original datasets are cleaned up to reflect records of Howard County only.


File Name
==========================================================================
LiquorLicense-Violence-Vehicle-Accident.xls


File Size
==========================================================================
1,236KB


Platform
==========================================================================
In the process of preparing this dataset, Open Data Portal of Howard County and the State of Maryland Open Data Portal are used. In addition, an online database is used to query liquor licenses provided in the county.


File
==========================================================================
Vehicle-collisions.xls
This file contains number of vehicle collisions in Howard County within specific location, time, days of the week and the road the collision happened.


Data Structure
==========================================================================
CASE_NUMBER, BARRACK ACC_DATE, ACC_TIME, ACC_TIME_CODE, DAY_OF_WEEK ROAD, INTERSECT_ROAD, DIST_FROM_INTERSECT, DIST_DIRECTION, CITY_NAME, COUNTY_CODE, COUNTY_NAME VEHICLE_COUNT, PROP_DEST, INJURY, COLLISION_WITH_1, COLLISION_WITH_2

File
==========================================================================
Police-Dept-Call-For-Liquor-Violation.xls
This file is about the service call made to police due to liquor violation and intoxication.


Data Structure
==========================================================================
Date_Reported, Time_Reported, Location, Statistical_Reporting_Area, Beat, Computer_Aided_Dispatch_Event_Type

File
==========================================================================
Liquor-License.xls
Name and location of business that are licensed for alcohol and tobacco in Howard County

Data Structure
==========================================================================
TaxRegNo, CorporateName, TradeAsName, StreetAddr1, StreetAddr2, City, County, ZipCode, Phone, LicenseClass, Licensee1, Licensee2, Licensee3, LastUpdate, Status


Proposed Research Question
==========================================================================
Does issuing more liquor license aggravate violence and vehicle accident in Howard County MD?
